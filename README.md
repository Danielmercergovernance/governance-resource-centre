# Governance Resource Centre

Welcome to the **Governance Resource Centre**, a growing collection of practical resources covering corporate governance, company secretarial practice, board support and governance recruitment within the UK.

The aim of this project is to provide accessible guidance for organisations, directors and governance professionals through clear, practical content.
